Stock Tracker API Configuration Guide

Configuration Setup

The Stock Tracker application uses a configuration file to manage API keys and other settings. This guide will help you set up the configuration properly.

Initial Setup

  1. Locate the config.template.js file in your project directory
  2. Create a copy of this file and name it config.js
  3. Update the values in config.js with your actual configuration
Important: Never commit your config.js file to version control as it contains sensitive information. The config.js file is already added to .gitignore.

Configuration Structure

The configuration file contains the following settings:

const config = {
    // API Keys
    STOCK_API_KEY: 'your_alphavantage_api_key_here', // Get from https://www.alphavantage.co/support/#api-key
    PERPLEXITY_API_KEY: 'your_perplexity_api_key_here', // Get from Perplexity AI
    
    // API Endpoints
    API_BASE_URL: 'https://www.alphavantage.co', // Alpha Vantage API base URL
    PERPLEXITY_API_URL: 'https://api.perplexity.ai/chat/completions', // Perplexity AI API endpoint
    
    // Other Configuration
    UPDATE_INTERVAL: 60000, // Stock data refresh interval in milliseconds (1 minute)
    MAX_STOCKS: 10,        // Maximum number of stocks that can be tracked
    
    // Feature Flags
    ENABLE_DEBUG: false,   // Enable/disable debug logging
    ENABLE_REAL_TIME: true, // Enable/disable real-time updates
    USE_FAKE_PP: false     // Use fake Perplexity API data for testing
};

// Make config available globally
window.config = config;

export default config;

Configuration Options

API Keys

  • STOCK_API_KEY: Your Alpha Vantage API key for accessing stock data
  • PERPLEXITY_API_KEY: Your Perplexity AI API key for AI-powered company search

API Endpoints

  • API_BASE_URL: The Alpha Vantage API base URL
  • PERPLEXITY_API_URL: The Perplexity AI API endpoint for company search

Other Configuration

  • UPDATE_INTERVAL: How often to refresh stock data (in milliseconds)
  • MAX_STOCKS: Maximum number of stocks that can be tracked simultaneously

Feature Flags

  • ENABLE_DEBUG: Enable/disable debug logging
  • ENABLE_REAL_TIME: Enable/disable real-time updates
  • USE_FAKE_PP: Toggle between real and fake Perplexity API data for testing

Browser Integration

The configuration is made available globally through the window object. You can access the configuration values in your JavaScript code using either:

// Using bracket notation (recommended)
window['config'].STOCK_API_KEY

// Using dot notation
window.config.STOCK_API_KEY
Note: Bracket notation is recommended for more reliable access to configuration values in the browser environment.

Getting API Keys

To use the Stock Tracker application, you'll need two API keys:

1. Alpha Vantage API Key

  1. Visit Alpha Vantage
  2. Sign up for a free API key
  3. Copy your API key and paste it into the STOCK_API_KEY field in your config.js file

2. Perplexity AI API Key

  1. Visit Perplexity AI
  2. Follow their documentation to obtain an API key
  3. Copy your API key and paste it into the PERPLEXITY_API_KEY field in your config.js file
